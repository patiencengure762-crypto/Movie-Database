## Movie Database Web App

A modern and responsive Movie Database web application that allows users to explore movies in a clean and interactive interface. The application displays movie information such as titles, posters, ratings, and descriptions, making it easy for users to discover and browse different films.


## Features

-  Browse a collection of movies
-  Search for movies easily
-  View movie ratings and descriptions
-  Display movie posters in a clean layout
-  Fully responsive design for mobile and desktop
-  Fast and lightweight frontend implementation

## Technologies Used

- HTML5 – Structure of the web pages
- CSS3 – Styling and responsive design
- JavaScript – Dynamic functionality and user interaction

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open index.html in your browser

The application will run instantly without requiring any installation.
